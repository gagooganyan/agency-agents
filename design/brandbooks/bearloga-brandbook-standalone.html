01 / О бренде и философии

Подход
к айдентике

Bearloga — дружелюбное гастро-пространство, где сочетаются сытная еда, душевная атмосфера и тёплая компания. Это место про посиделки, вкусные бургеры, лапшу, пиццу и коктейли — по-домашнему, но с характером.

Концепция строится на сочетании брутального шрифта и контурных, «ручных» иллюстраций, как будто нарисованных на стене или салфетке за ужином. В графике появляются медвежьи следы — отсылки к имени и образу «берлоги».

02 / Логотип

Гибкая

Шрифтовой логотип BEARLOGA выполнен в кастомном гротеске: буквы вырезаны из теста или дерева. Разработаны горизонтальная и вертикальная версии для адаптации под разные форматы.

03 / Маскот

Медведь

Медведь-маскот — дружелюбный, тёплый персонаж, который подчёркивает идею уюта, «логова» и душевной еды. Одновременно напоминает настоящего зверя и плюшевую игрушку — символ мягкой дикости и спокойной силы.

🐾
Добродушный, но с характером Не агрессивный, но уверенный в себе
🧸
Мягкая дикость Между настоящим зверем и плюшевой игрушкой
🏠
Тепло и притяжение Mascot = визуальный якорь тёплой берлоги
📐
Гибкий модуль Целиком или фрагментарно (только голова в аватаре)

06 / Иллюстрации

Ручной
стиль

Иллюстрации — сердце визуального языка Bearloga. Мягкая линия, как угольный карандаш. Округлые, массивные «медвежьи» формы. Простые, добрые люди, которые едят, смеются и радуются компании.

👥
Персонажи

Простые добрые люди с медвежьими очертаниями — округлые фигуры, мохнатые локти. Всегда заняты едой и общением.

🤲
Руки и еда

Крупные «медвежьи лапы» держат бургер, лапшу, кофе, пиццу. Создают ощущение тепла и заботы.

🍔
Иконки еды

Бургер, стакан, пицца, миска, чашка, вилка — лаконичные контурные рисунки для меню и навигации.

10 / Правила использования

Можно
и нельзя

✓ Логотип — можно
✓ Использовать на фонах из палитры бренда
✓ Менять цвет логотипа согласно таблице сочетаний
✓ Масштабировать с сохранением пропорций
✓ Использовать обе версии (горизонтальная / вертикальная)
✓ Применять монохромную версию на белом
✕ Логотип — нельзя
✕ Искажать, растягивать, деформировать
✕ Добавлять тени, градиенты, свечение
✕ Помещать на пёстрые фото без подложки
✕ Использовать цвета вне палитры
✕ Уменьшать ниже 30 мм / 120 px
✓ Типографика — можно
✓ Firenight — только капслок для заголовков
✓ Golos — основной текст в любом регистре
✓ Swanky — одна короткая акцентная фраза
✓ Использовать не более двух шрифтов на носитель
✕ Типографика — нельзя
✕ Firenight в строчном регистре
✕ Заменять системными шрифтами (Arial, Helvetica)
✕ Swanky and Moo как основной текстовый шрифт
✕ Смешивать больше двух шрифтов на носителе

13 / CSS Design Tokens

Токены для
цифровой
команды

Подключить как первый файл в проекте. Все цвета, шрифты, отступы и анимации — из этих переменных.

:root {
  /* ─── ЦВЕТА ─── */
  --bearloga-color-campfire:   #E8491E;   /* Жар костра */
  --bearloga-color-cream:      #F2E4C0;   /* Тёплый кремовый */
  --bearloga-color-woody:      #2D0E06;   /* Древесный */
  --bearloga-color-forest:     #3A4B1B;   /* Лесной зелёный */
  --bearloga-color-charcoal:   #1E1C1A;   /* Угольный след */
  --bearloga-color-white:      #FFFFFF;

  /* Семантика */
  --bearloga-color-accent:      var(--bearloga-color-campfire);
  --bearloga-color-bg-primary:  var(--bearloga-color-cream);
  --bearloga-color-bg-dark:     var(--bearloga-color-woody);
  --bearloga-color-text:        var(--bearloga-color-charcoal);

  /* Hover */
  --bearloga-color-campfire-hover: #C93D17;
  --bearloga-color-forest-hover:   #2E3C14;

  /* ─── ШРИФТЫ ─── */
  --bearloga-font-headline: 'Firenight', 'Impact', sans-serif;
  --bearloga-font-body:     'Golos Text', 'Inter', Arial, sans-serif;
  --bearloga-font-accent:   'Swanky and Moo Moo', cursive;

  /* ─── РАЗМЕРЫ ─── */
  --bearloga-text-sm:   0.875rem;   /* 14px */
  --bearloga-text-base: 1rem;       /* 16px */
  --bearloga-text-xl:   1.5rem;     /* 24px */
  --bearloga-text-3xl:  2.5rem;     /* 40px */
  --bearloga-text-5xl:  5rem;       /* 80px */

  /* ─── ОТСТУПЫ ─── */
  --bearloga-space-section: 4rem;
  --bearloga-space-block:   2rem;
  --bearloga-space-element: 1rem;

  /* ─── РАДИУСЫ ─── */
  --bearloga-radius-md:   8px;
  --bearloga-radius-lg:   12px;
  --bearloga-radius-pill: 9999px;

  /* ─── ТЕНИ (тёплые) ─── */
  --bearloga-shadow-md:
    0 4px 6px rgba(45,14,6,0.10),
    0 2px 4px rgba(45,14,6,0.06);
  --bearloga-shadow-accent:
    0 4px 16px rgba(232,73,30,0.30);

  /* ─── АНИМАЦИЯ ─── */
  --bearloga-ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
  --bearloga-duration-base: 250ms;
}
